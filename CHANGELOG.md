# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-24
### Added - Phase 2 Final Release
- Firebase Authentication (Register, Login, Logout, Forgot Password)
- Product listing from Firestore with category filter
- Wishlist with Firestore persistence
- Shopping cart (add, remove, update quantity)
- Checkout with delivery address input
- Order placement and order history screen
- User profile view and edit
- Onboarding screen
- MVVM architecture with Provider
- Custom animated product cards
- Release APK v1.0.0
- Video demonstration published on YouTube

## [0.1.0] - 2026-05-20
### Added - Phase 1 UI Only
- Figma design completed
- All screens implemented in Flutter
- Navigation between screens
- Responsive layouts
- No Firebase at this stage
